Frequently Asked Questions

What file formats are supported?

RevealAI supports MP4, AVI, MOV, and MKV for video; WAV, MP3, M4A, and FLAC for audio. Maximum file size is 500 MB.

How long does analysis take?

Most analyses complete within 1-3 minutes depending on file size and server load. You'll see a progress bar indicating processing status.

What does the score mean?

Scores range from 0-100%. Below 30% indicates authentic content, 30-70% is uncertain (manual review recommended), and above 70% indicates likely deepfake.

Can I download the analysis report?

Yes! After analysis completes, you can download a detailed PDF report including heatmaps, scores, and technical analysis.
