User Guide

Learn how to use RevealAI to detect deepfakes in video and audio content

Getting Started

RevealAI is a professional deepfake detection system that analyzes video and audio files to determine if they contain synthetic content. Follow these steps to get started:

  1. Navigate to Home: Click "Home" in the navigation menu to go to the main upload page
  2. Select a File: Choose a video (MP4, AVI, MOV) or audio file (WAV, MP3)
  3. Click Analyze: Press the "Analyze Media" button to start processing
  4. Wait for Results: Processing takes 30-60 seconds. A progress indicator will show the status
  5. Review Results: View your analysis results including scores and verdicts

Supported File Formats

Video Files

  • - MP4 (.mp4)
  • - AVI (.avi)
  • - MOV (.mov)
  • - MKV (.mkv)

Maximum file size: 500 MB recommended

Audio Files

  • - WAV (.wav)
  • - MP3 (.mp3)
  • - M4A (.m4a)
  • - FLAC (.flac)

Maximum file size: 500 MB recommended

Understanding Results

Verdict Color Codes

Authentic (Green)

Content is likely genuine. Score below 30%

Uncertain (Yellow)

Results inconclusive. Score 30-70%

Deepfake (Red)

Content likely synthetic. Score above 70%

Score Explanation

RevealAI provides three scores for your media:

  • Video Score: Analysis of visual content (0-100%)
  • Audio Score: Analysis of audio content (0-100%)
  • Combined Score: Final verdict combining both analyses

System Accuracy: Video 95%, Audio 92%, Combined 94%

Heatmaps and Visualizations

For video analysis, you will see heatmap overlays showing regions of the video that contributed most to the deepfake probability. Bright red areas indicate higher suspicion.

Advanced Features

PDF Reports

After analysis, you can download a detailed PDF report containing:

  • - Complete analysis summary
  • - All scores and confidence metrics
  • - Heatmap visualizations
  • - System methodology explanation
  • - Final verdict and recommendations

FAQ

What is a deepfake?

A deepfake is synthetic media created using AI and machine learning where a person in a video or image is replaced with someone else. RevealAI detects these synthetic replacements.

How accurate is RevealAI?

RevealAI achieves 95% accuracy on video analysis, 92% on audio analysis, and 94% combined accuracy. However, as deepfake technology improves, accuracy may vary with newer techniques.

Can I analyze only audio or only video?

Yes! You can upload either a video file or an audio file separately. The system will analyze whichever media type you provide.

Why did analysis fail?

Analysis may fail if the file is corrupted, in an unsupported format, or too large. Make sure your file is under 500 MB and in a supported format (MP4, WAV, etc.).

Is my uploaded content stored?

Files are processed temporarily and deleted after analysis completes. No permanent storage of your media files occurs.

What should I do with the results?

Use RevealAI results as a tool to inform your decision-making. For important decisions, combine RevealAI analysis with other verification methods and expert review.

Tips for Best Results

  • Use clear, high-quality video or audio files
  • Ensure adequate lighting in videos
  • Provide at least 10 seconds of content for reliable analysis
  • Test with known authentic and deepfake samples to understand the scoring
  • Remember that uncertain results may require expert review
  • Keep the file size under 500 MB for best performance
